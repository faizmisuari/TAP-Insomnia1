Welcome to the Insomnia Test Project

This project demonstrates API testing using Insomnia.

Getting Started

  1. Clone the repository to your local machine.
  2. Install Insomnia if not already installed.
  3. Open Insomnia and import the provided API tests.
  4. Configure any environment variables or settings.
  5. Run API tests to validate endpoints and responses.

For more detailed information, refer to the project's README file.
